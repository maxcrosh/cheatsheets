### Get started with Django
https://www.djangoproject.com/


#### 1. Install framework
```console
pip install Django
```


#### 2. Create new project
```console
django-admin startproject mysite
```


#### 3. Project folder:
```
mysite
├── manage.py
├── mysite
│   └── __init__.py 
│   └── settings.py
│   └── urls.py
│   └── asgi.py
│   └── wsgi.py 
````


#### 4. Run web server
```console
python manage.py runserver
```
